Remise à niveau

Ce bloc est conçu pour construire une base solide, essentielle avant d'aborder les concepts plus avancés de la formation.

  • SQL (B4U21420) : Apprenez les fondamentaux des bases de données relationnelles. Ce cours couvre la création et la manipulation de tables, les requêtes complexes pour extraire des données, l'utilisation des jointures et des fonctions d'agrégation.
  • Python (B4U20320) : Une introduction au langage de programmation Python, en mettant l'accent sur les structures de données, la syntaxe de base, le contrôle de flux et la création de fonctions. Les étudiants apprendront à écrire des scripts simples et efficaces.
  • Introduction à Java : Ce cours présente les concepts de la programmation orientée objet avec Java. Il aborde les bases du langage, les classes, les objets, l'héritage et le polymorphisme, des notions clés pour les projets de développement.
  • Statistique (B4U20920) : Découvrez les principes de la statistique descriptive et inférentielle. Le cours inclut l'analyse de données, les tests d'hypothèses, les probabilités et les bases de la régression, pour apprendre à interpréter correctement les données.

Apprentissage de Python

Ce bloc se concentre sur l'application de Python à des tâches concrètes de gestion et d'analyse de données.

  • Data Management, Dataviz, Data Mining (B4U20522) : Utilisez les bibliothèques Python comme Pandas et Numpy pour manipuler, nettoyer et analyser des ensembles de données volumineux. Vous apprendrez aussi à créer des visualisations de données percutantes (dataviz) et à extraire des tendances (data mining).
  • Python avancé (Web Scrapping + API) (B4U20522) : Plongez dans les techniques de collecte de données sur le web. Ce cours vous formera au web scraping pour extraire des informations à partir de sites web, et à l'utilisation des API pour interagir avec des services externes et récupérer des données structurées.

Cloud

Ce bloc est dédié aux technologies de calcul distribué et aux bases de données modernes, essentielles dans le monde du Big Data.

  • Framework de calcul distribué/spark (Cloud 1) (B4U20822) : Ce cours vous initiera aux principes du calcul distribué et à l'utilisation de Apache Spark pour le traitement de données à grande échelle.
  • Développement Java Neo4J (Cloud 2) (B4U20722) : Apprenez à concevoir et à interagir avec des bases de données orientées graphe à l'aide de Neo4J et de Java. Ce cours est axé sur la modélisation et l'interrogation de données complexes.
  • Data Streaming (Cloud 3) (B4U21022) : Explorez les concepts du traitement de données en temps réel. Ce cours couvre les technologies et les architectures nécessaires pour ingérer, traiter et analyser des flux de données continus.

Modélisation et CI/CD

Ce bloc est axé sur la création, l'évaluation et le déploiement de modèles d'apprentissage automatique.

  • Machine Learning 1 (B4U21322) : Une introduction au Machine Learning couvrant les algorithmes de base pour la classification, la régression et le clustering. Vous apprendrez à construire vos premiers modèles prédictifs.
  • Machine Learning 2 (B4U21522) : Approfondissez les concepts du Machine Learning avec des algorithmes plus avancés et des techniques d'évaluation de modèles. Ce cours se concentre sur la pratique et l'optimisation des performances.
  • Deep Learning (B4U20422) : Plongez dans les réseaux de neurones. Ce cours vous enseignera à concevoir des modèles de Deep Learning pour des tâches complexes comme la reconnaissance d'images ou le traitement du langage naturel.
  • ML Ops (B4U20222) : Apprenez à gérer le cycle de vie des projets de Machine Learning, du développement à la mise en production. Ce cours couvre les outils et les pratiques de l'intégration continue et du déploiement continu (CI/CD) dans le contexte du Machine Learning.
  • Generative AI (B4U20622) : Découvrez les modèles d'IA Générative et leur fonctionnement. Vous explorerez les applications des modèles de langage et d'image, ainsi que les principes de leur entraînement et de leur utilisation.

Séminaire Pro

Ce bloc est une préparation directe au monde professionnel, en se concentrant sur les outils et les compétences pratiques.

  • Séminaire Pro "PowerBI" (B4U21120) : Maîtrisez l'outil de Business Intelligence de Microsoft. Ce séminaire vous apprendra à créer des tableaux de bord interactifs et des rapports analytiques pour une prise de décision basée sur les données.
  • Séminaire Pro (B4U21420) : Un séminaire généraliste axé sur le développement des compétences professionnelles, la présentation de projets et la préparation aux soutenances.
