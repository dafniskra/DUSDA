Remise à niveau

Ce bloc est conçu pour construire une base solide, essentielle avant d'aborder les concepts plus avancés de la formation.

  • SQL (B4U21420) : Apprenez les fondamentaux des bases de données relationnelles.
  • Python (B4U20320) : Une introduction au langage de programmation Python, en mettant l'accent sur les structures de données, la syntaxe de base, le contrôle de flux et la création de fonctions.
  • Introduction à Java : Ce cours présente les concepts de la programmation orientée objet avec Java.
  • Statistique (B4U20920) : Découvrez les principes de la statistique descriptive et inférentielle.

Apprentissage de Python

Ce bloc se concentre sur l'application de Python à des tâches concrètes de gestion et d'analyse de données.

  • Data Management, Dataviz, Data Mining (B4U20522) : Utilisez les bibliothèques Python comme Pandas et Numpy pour manipuler, nettoyer et analyser des ensembles de données volumineux.
  • Python avancé (Web Scrapping + API) (B4U20522) : Plongez dans les techniques de collecte de données sur le web. Ce cours vous formera au web scraping et à l'utilisation des API.

Cloud

Ce bloc est dédié aux technologies de calcul distribué et aux bases de données modernes, essentielles dans le monde du Big Data.

  • Framework de calcul distribué/spark (Cloud 1) (B4U20822) : Ce cours vous initiera aux principes du calcul distribué et à l'utilisation de Apache Spark pour le traitement de données à grande échelle.
  • Développement Java Neo4J (Cloud 2) (B4U20722) : Apprenez à concevoir et à interagir avec des bases de données orientées graphe à l'aide de Neo4J et de Java.
  • Data Streaming (Cloud 3) (B4U21022) : Explorez les concepts du traitement de données en temps réel.

Modélisation et CI/CD

Ce bloc est axé sur la création, l'évaluation et le déploiement de modèles d'apprentissage automatique.

  • Machine Learning 1 (B4U21322) : Une introduction au Machine Learning couvrant les algorithmes de base pour la classification, la régression et le clustering.
  • Machine Learning 2 (B4U21522) : Approfondissez les concepts du Machine Learning avec des algorithmes plus avancés.
  • Deep Learning (B4U20422) : Plongez dans les réseaux de neurones pour des tâches complexes comme la reconnaissance d'images.
  • ML Ops (B4U20222) : Apprenez à gérer le cycle de vie des projets de Machine Learning et l'intégration continue et du déploiement continu (CI/CD).
  • Generative AI (B4U20622) : Découvrez les modèles d'IA Générative et leurs applications.

Séminaire Pro

Ce bloc est une préparation directe au monde professionnel, en se concentrant sur les outils et les compétences pratiques.

  • Séminaire Pro "PowerBI" (B4U21120) : Maîtrisez l'outil de Business Intelligence de Microsoft.
  • Séminaire Pro (B4U21420) : Un séminaire généraliste axé sur le développement des compétences professionnelles.

Mémoires

Le mémoire est l'aboutissement de votre parcours, une occasion unique de prouver votre capacité à mener un projet de recherche de A à Z. Vous avez deux options :

1. Proposer votre propre sujet

Vous pouvez choisir un sujet qui vous passionne, en accord avec l'un de nos professeurs. C'est l'opportunité de vous spécialiser sur un thème qui vous intéresse particulièrement.

2. Choisir un sujet proposé

La faculté met à votre disposition une liste de sujets de recherche. Vous pouvez choisir un sujet parmi cette liste et être encadré par un expert du domaine.
